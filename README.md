# 📝 Todo Application

A simple and responsive Todo List web application built using HTML, CSS, and JavaScript.

## 🚀 Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Clean and minimal user interface
- Dynamic DOM manipulation

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript
